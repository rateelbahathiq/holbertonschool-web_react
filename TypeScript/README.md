# TypeScript Project - Holberton School

This repository contains a series of tasks designed to learn and practice **TypeScript fundamentals and advanced concepts**.


---

## 📁 Project Structure

```
TypeScript/
├── task_0/
├── task_1/
├── task_2/
├── task_3/
├── task_4/
└── task_5/
```

---

## 🚀 Tasks Overview

### 🔹 Task 0 - Basic Types & Interfaces

* Created a `Student` interface
* Used TypeScript types for variables
* Rendered a table using Vanilla JavaScript

---

### 🔹 Task 1 - Advanced Interfaces & Classes

* Created `Teacher` interface
* Used `readonly` and optional properties
* Extended interfaces (`Directors`)
* Created function interfaces
* Implemented a class with constructor interfaces

---

### 🔹 Task 2 - Classes & Type Narrowing

* Implemented `Director` and `Teacher` classes
* Used union types (`number | string`)
* Created type predicate (`isDirector`)
* Implemented function behavior based on type (`executeWork`)
* Used string literal types (`Subjects`)

---

### 🔹 Task 3 - Ambient Namespaces

* Created type and interface (`RowID`, `RowElement`)
* Used `.d.ts` files for external JavaScript typing
* Integrated a simulated CRUD library

---

### 🔹 Task 4 - Namespaces & Declaration Merging

* Used TypeScript namespaces (`Subjects`)
* Implemented declaration merging for interfaces
* Built multiple subject classes (`Cpp`, `React`, `Java`)
* Managed optional properties and inheritance

---

### 🔹 Task 5 - Nominal Typing (Branding)

* Implemented branded interfaces (`MajorCredits`, `MinorCredits`)
* Ensured type safety using unique identifiers
* Created functions to safely operate on branded types

---

## 🧠 Concepts Covered

* TypeScript types & interfaces
* Optional and readonly properties
* Function typing
* Classes and constructors
* Inheritance and extension
* Union types
* Type predicates
* Namespaces
* Declaration merging
* Ambient declarations (`.d.ts`)
* Nominal typing (branding)
